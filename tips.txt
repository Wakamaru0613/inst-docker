dockerをまとめてビルド・起動 -dはバックグラウンド実行
docker-compose up --build -d

プロジェクトフォルダは例のごとく英数字のみ

postgreのdataディレクトリをvolumes指定するとエラーになる

起動後
nodeサーバで
#接続
docker exec -it node-dev /bin/sh
#express作成 /appはプロジェクト名
express -f --view=ejs /app
npm install -g jquery --save
npm install -g bootstrap@4 --save
npm install -g pg --save
npm install


#起動
npm start
